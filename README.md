### Prerequisites
*   Node.js (v18+)
*   npm

### Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Configuration
1.  Create a `.env` file in the root directory.
2.  Add your Firebase configuration keys (see `.env.example` if available).

### Running Locally
```bash
npm run dev
```

### Deployment
Built with Vite and Firebase.
```bash
npm run build
firebase deploy
```
